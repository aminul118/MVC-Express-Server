## MVC Format Backend server Express

#### Post a food in this route

```
http://localhost:5000/api/foods
```

#### Get all foods

```
http://localhost:5000/api/foods
```

#### Get indivisal food by _id

```
http://localhost:5000/api/food/${_id}
```

Example

```
http://localhost:5000/api/food/67a8fca933bea00e5f82a6e3
```
